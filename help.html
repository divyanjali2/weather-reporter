PHP Utility Functions Documentation

1. formatDate($date, $format = 'd M Y')

This function formats a given date string into a specific format.

Parameters:

  • $date (string) - The date to be formatted. It can be in any valid date format.
  • $format (string, optional) - The desired output date format (default: 'd M Y').

Return:

A formatted date string based on the provided format.

Example:

$formattedDate = formatDate('2025-03-27', 'Y-m-d'); // Output: 2025-03-27

2. truncate($text, $length = 100, $suffix = '...')

This function truncates a string to a specified length and appends a suffix (default is '...').

Parameters:

  • $text (string) - The text to truncate.
  • $length (int, optional) - The maximum length of the string (default: 100).
  • $suffix (string, optional) - The suffix to append (default: '...').

Return:

A truncated string with the specified suffix.

Example:

$truncatedText = truncate('This is a very long sentence that needs to be shortened.', 20); // Output: 'This is a very lo...'

3. verifyRecaptcha($recaptcha_response)

This function verifies a Google reCAPTCHA response to ensure that the user is human.

Parameters:

  • $recaptcha_response (string) - The reCAPTCHA response token from the client-side.

Return:

A boolean indicating whether the reCAPTCHA validation was successful (true) or not (false).

$isValid = verifyRecaptcha($recaptchaResponseToken); // Output: true or false

4. fetch($url, array $options = [])

This function makes HTTP requests to APIs using cURL and returns the API response.

Parameters:

  • $url (string) - The API URL to send the request to.
  • $options (array, optional) - Configuration options (default: empty array). Available options:
    • method (string) - The HTTP method (default: 'GET').
    • headers (array) - Any custom headers to be included in the request.

Return:

An array containing the HTTP status code and the decoded response data (or the raw response).

Example:

$response = fetch('https://jsonplaceholder.typicode.com/posts'); // Returns a response array with status and data
